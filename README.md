## Installation

### Alacritty

Terminal emulator that supports
full RGB range

**Requirements**
- Font: `Roboto Mono` [Link](https://github.com/googlefonts/RobotoMono.git)

```shell
ln -s ~/.config/dotfiles/alacritty ~/.config/alacritty 
```
